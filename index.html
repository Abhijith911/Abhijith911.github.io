01
Mini Project Raspberry Pi 4 OpenCV Tesseract OCR Flask ESP8266
Vision-Based Smart Parking System
↗

Fully automated parking system deployed on Raspberry Pi 4. Pi Camera V2 captures vehicle frames via a GStreamer libcamera pipeline; OpenCV detects number plates using Canny edge detection, contour analysis, and aspect ratio filtering; Tesseract OCR extracts and validates Indian plate numbers.

ESP8266 firmware reads HC-SR04 ultrasonic sensor with a 2.5-second confirmation filter and sends VEHICLE_DETECTED over UART to the Pi, triggering slot confirmation via PySerial. A Flask dashboard displays live slot status, estimated fees, overstay alerts, reservation, blacklist/whitelist, and a waiting queue — all auto-refreshing every 5 seconds.
